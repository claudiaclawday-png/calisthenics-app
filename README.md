# Seguimiento de Calistenia

Una aplicación web progresiva (PWA) para llevar el control de tu rutina de calistenia.

## Características

- Seguimiento de rutinas de dominadas y fondos
- Tres tipos de entrenamiento: Max Reps, Sub Max y Volumen Escalera
- Temporizadores para descansos
- Historial de entrenamientos
- Instalable como PWA en dispositivos móviles

## Tecnologías

- Next.js
- React
- Tailwind CSS
- Zustand para gestión de estado

## Requisitos

- Node.js 18 o superior
- npm o yarn
